# 스프링부트 기능정의
* 단독 실행 가능한 스프링 애클리케이션 생성
* 내장 컨테이너로 톰캣, 제티, 언더토우 중에서 선택가능
* 스타터를 통한 간결한 의존성 구성 지원
* 스프링에 대한 자동구성 제공
* 더이상 xml구성 필요없음
* 제품출시 후 운영에 필요한 다양한 기능(상태점검, 모니터링 등) 제공

# 스프링 부트 구성요소
* 빌드도구(그레이들 vs 메이븐)
* 스프링 프레임워크
* 스프링 부트
* 스프링 부트 스타터
