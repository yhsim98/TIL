# Min Algorithm
* 앞으로 가장 오랫동안 참조하지 않을 page 교체
* 실현불가능함
    * page referene string을 알아야 한다
* 그래서 교체 기법의 성능 평가 도구로 활용됨

# Random Algorithm
* 무작위로 교체할 page 선택
* Low overhead
* No policy
