# 의존성
## 클래스 의존성의 종류
* 연관관계
    * A에서 B로 갈 수 있는 경우
    * 멤버변수로 B를 가지고 있다
* 의존관계
    * A가 파라미터나 리턴타입으로 B를 가지는 경우
    * 특정 시점에만 관계를 맺는 경우
* 상속관계
    * 부모객체의 구현이 바뀌면 자식들은 영향을 받는다
* 실체화 관계
    * 인터페이스 구현
