# 배치 프로그램
사용자와의 상호 작용 없이 여러 작업들을 미리 정해진 일련의 순서에 따라 일괄적으로 처리하는 것을 의미한다.

* 정기 배치 : 정기적으로 배치
* 이벤트성 배치 : 조건
* On-Demand : 사용자 요청 시 수행

배치 프로그램이 갖추어야 할 필수 요소
* 대용량 데이터 처리 가능
* 자동화 가능
* 견고성
* 안정성/신뢰성 : 오류 발생하면 추적 가능
* 성능 : 다른 프로그램 방해하면 안됨, 속도
